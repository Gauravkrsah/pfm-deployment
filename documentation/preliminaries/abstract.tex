\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

\vspace{1cm}

\textbf{Personal Finance Manager (PFM)} is an AI-powered hybrid mobile application designed to revolutionize personal expense tracking through natural language processing. Unlike traditional finance management tools that require tedious manual data entry and complex form-filling, PFM enables users to record transactions using simple natural language inputs such as ``biryani rahul 500'' or ``taxi to airport 1500,'' from which the system automatically extracts the amount, categorizes the expense, and identifies any associated parties. Built on a modern technology stack combining a \textbf{React} frontend with a \textbf{FastAPI} backend, the application leverages \textbf{Google Gemini AI} for intelligent parsing and \textbf{Supabase} for robust data persistence. The system features a hybrid NLP approach that utilizes over 50 regex patterns alongside AI-powered parsing to ensure high accuracy and responsiveness across 20+ expense categories. Key capabilities include multi-group management for personal and shared expenses, comprehensive loan tracking for money lent and borrowed, and interactive visual analytics for spending insights. Furthermore, a Retrieval-Augmented Generation (RAG) system enables users to query their financial data using natural language questions, providing contextually aware responses. This project demonstrates the effective integration of advanced AI technologies into practical financial applications, delivering a seamless cross-platform experience via \textbf{Capacitor} for native Android deployment.
