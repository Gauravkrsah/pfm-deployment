# Project Report Documentation

This directory contains the source files for the **Personal Finance Manager (PFM)** project report, formatted according to Tribhuvan University standards.

## Files

*   `FULL_REPORT.md`: The complete report in a single file. You can convert this to PDF using a Markdown-to-PDF converter (e.g., VS Code extension, Pandoc).
*   `00_Preliminaries.md`: Title Page, Approval, Abstract, TOC.
*   `01_Introduction.md`: Chapter 1 (Introduction, Problem Statement, Objectives).
*   `02_Literature_Review.md`: Chapter 2 (Existing Systems, Gap Analysis).
*   `03_System_Methodology.md`: Chapter 3 (SDLC, Requirements).
*   `04_System_Design.md`: Chapter 4 (Architecture, UML Diagrams).
*   `05_Implementation.md`: Chapter 5 (Tools, Code Snippets, Testing).
*   `06_Conclusion.md`: Chapter 6 (Conclusion, Future Work, References).

## Diagrams

The UML diagrams in Chapter 4 are written in **Mermaid.js** syntax. They should render automatically in GitHub, GitLab, or any Markdown viewer with Mermaid support.

## How to Submit

1.  Open `FULL_REPORT.md` in a Markdown editor (like VS Code or Obsidian).
2.  Export to PDF.
3.  Ensure the page breaks for Title Page, Certificate, etc., are correct (you may need to insert manual page breaks `<div style="page-break-after: always;"></div>` if your converter requires them, though standard converters handle headers well).
